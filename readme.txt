# Mork
Minimal HTML5 Template.

## About
Mork is perfect for building portfolio sites, for agencies, individuals and more... Mork is simple, clear, focused on modern appearance and ease of customization.

## Template Overview
* Fully Responsive Layout
* Modern Design
* HTML5 & Bootstrap 4
* CSS3 and jQuery Smooth Animations
* W3C HTML Valid Code
* 2 Home Page Variants
* Elegant and Minimal HTML Pages
* Mobile Touch Friendly Slider
* Easy to Customize Code
* Google Maps Integration
* Working Contact Form
* Google Web Fonts
* Cross Browser Support
* Well Documented
    
## Recommended requirements
* Mork supports only the latest, stable releases of all major browsers and platforms.
